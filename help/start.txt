<div>
	<h1>Welcome to Tuchulcha</h1>
	<p>
		You can always open this document selecting "Introduction" in
		the Help menu.
	</p>

	<h2>Start</h2>
	<p>
		To create a new (empty) file, select file->new or type
		CTRL+N.<br />
		To open an existing file, select file->open or type
		CTRL+O.<br />
	</p>
	<h2>Editing a Document</h2>
	<p>
		If you have an open document, you can add new items by
		drag&drop (choose one from the selector widget and drag it onto
		the document) or by double click on the item to add.
		You will be asked for a valid object name.
		If you click at an item in the selector widget,
		the corresponding class documentation will appear here in the
		help browser.
	</p>
	<p>
		Connections between objects can also be established by
		drag&drop, simply drag the source slot onto the target slot.
		If an input slot is already connected, it will be disconnected
		before establishing the new connection.
	</p>
	<p>
		Objects can be deleted pressing the DEL key and renamed using
		F2 or by double click. In either case, a dialog will ask for
		your confirmation.
	</p>
	<p>
		Object parameters can be changed using the objectIspector
		widget. Tooltips show a short documentation for each parameter.
	</p>
</div>
